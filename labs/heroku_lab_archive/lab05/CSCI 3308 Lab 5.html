Part A: Section 1. Javascript Tutorial

(15 Minutes)

  1. Using Javascript to change HTML content
    2. Client-side javascript can make HTML pages more dynamic and interactive. How does javascript know which HTML element to change? Let's look at an example. 
    
<!DOCTYPE html>
<html>
<body>

	<button class="btn btn-primary" onclick="document.getElementById('demo').innerHTML = Date();"> Display Date and Time.</button>
	<p id="demo"></p>

</body>
</html>
    Copy


    onclick="document.getElementById('demo').innerHTML = Date()"
    document.getElementById('demo').innerHTML = Date() is the javascript code that will execute "onclick" of the button
    document.getElementById('demo')
    The "document" object refers to the HTML document which contains all of the content on the page. "getElementById()" is a function that retrieves the HTML element with a specific id, in this case "demo"
    innerHTML
    "innerHTML" is the element property that contains the HTML markup within the element
    Date()
    "Date()" is a javascript object that represents the day and time. Javascript stores dates as milliseconds since January 01, 1970, 00:00:00 UTC (Universal Time Coordinated)
    id="demo"
    This line allows us to tag the paragraph element so that we can reference it inside of our javascript code. Remember that element ids have to be unique.

  2. HTML <script> tag
    3. Placing our javascript code inside of our HTML tags can be messy, especially for more complex javascript code. We can place our javascript code inside a separate set of <script> tags like this: 
    
<!DOCTYPE html>
<html>
	<script>
		function displayDate(id) {
			document.getElementById(id).innerHTML = Date();
		}
	</script>
<body>

	<button class="btn btn-primary" onclick="displayDate('demo2');"> Display Date and Time.</button>
	<p id="demo2"></p>

</body>
</html>
    Copy


    onclick="displayDate(demo2)"
    Our "onclick" method now calls a function named "displayDate()" and we are passing in the string "demo2"
    function displayDate(id)
    This line defines our new function that takes one parameter, id
    document.getElementById(id).innerHTML = Date();
    This line is pretty much the same as before. We have simply moved it into a function.

  3. Javascript in a separate file
    4. You can place your javascript code inside the HTML <script> tags or you can place your javascript code inside a separate file. How do we link the javascript and HTML files? We can link our javascript file to our HTML file by placing a reference in our HTML file like so: 
    
<script src="../resources/js/my_scripts.js"></script>

    Calling a Function With Parameters

    In the previous example, we called a method with no parameters. But we don't want to make an individual method for every option on our website! Instead, we'll want to look at using parameters to help us reuse our code and make our javascript code base more maintainable.

    <button class="btn btn-primary" onclick="updateValueByNum(5)">Increase By 5</button>
<button class="btn btn-primary" onclick="updateValueByNum(-5)">Decrease By 5</button>
<span id="changeNumber">0</span>
<script>
	function updateValueByNum(changeBy){
		var number = document.getElementById("changeNumber").innerHTML;//retrieve the current number
		number= parseInt(number) + changeBy;//Increase/Decrease by passed in parameter.
		document.getElementById("changeNumber").innerHTML = number;//Update to show the new number value
	}
</script>
    Copy

    Displayed as:       0

    More information
    onclick="updateValueByNum(5)"
    When we call a method that requires a parameter, we must provide the value it needs AND ensure it is the expected data type. Javascript isn't a compiled language like C# or Java, so you won't know until runtime if your values are correct. So make sure you use numeric or text parameters correctly.
    parseInt(number) + changeBy
    Because javascript doesn't have explicit data types, it tries to automatically determine the most appropriate datatype at run time. Unfortunately it can often be wrong! In our example here, we have used parseInt() to explicitly tell javascript that our parameter should be treated as an integer. Why? Because by default javascript will first treat our innerHTML data as as text information (a String). This means our "+" operator would first be interpreted as a concatenation not an addition! So we will use one of the parse methods to tell javascript how to correctly interpret our data.

    Conditional Logic

    In javascript, conditional logic uses a syntax similiar to C#, C++, and Java. In the example below, we've update our updateValueByNum() method to now have an upper & lower bound of 10 & -10. 

    <button class="btn btn-primary" onclick="updateValueByNumBounded(5)">Increase By 5</button>
<button class="btn btn-primary" onclick="updateValueByNumBounded(-5)">Decrease By 5</button>
<span id="changeNumberBounded">0</span>
<script>
	function updateValueByNumBounded(changeBy){
		var number = document.getElementById("changeNumberBounded").innerHTML;//retrieve the current number
		number= parseInt(number) + changeBy;//Increase/Decrease by passed in parameter.
		if(number > 10)
		{//Ensure we don't go above 10
			number = 10;
		}
		else if(number < -10)
		{//Ensure we don't go below -10
			number = -10;
		}
		document.getElementById("changeNumberBounded").innerHTML = number;//Update to show the new number value
	}
</script>

    Javascript conditional logic syntax requires you to write the conditional statement in between a set of paranethsis. This is identical to Java/C++/C# syntax. If you are a Python programmer, please note that the curly braces are required but tabbing/spacing is only encouraged (not a requirement by the interpreter).
    else if(number < -10)
    Similiar syntax to the "if statement", the "else if" will only be evaluated when the "if statement" results in a false statement.

    Modifying Style

    Instead of just modfiying the values/content of an html page, we can also modify the pages look and feel via updates to its css! The example below will increase and decrease the width of a horizontal bar

    <button class="btn btn-primary" onclick="changeWidth('increase')">Increase Width</button>
<button class="btn btn-primary" onclick="changeWidth('decrease')">Decrease Width</button>
<div id="barWidthExample" style="background-color:green;color:white;text-align:center;width:20%;">Bar!</div>
<script>
	function changeWidth(direction){
		var width = document.getElementById("barWidthExample").style.width;//retrieve the current width
		width = parseInt(width.replace('%', ''));//Remove the % percent sign from our width value
												 //and convert it to an integer
		if(direction == 'increase')
		{//Increase our width
			width = width + 5;
		}
		else if(direction == 'decrease')
		{//Decrease our width
			width = width - 5;
		}
		if(width > 100 || width <= 0)
		{//If our width becomes too large/small, reset to our default width
			width = 20;
		}
		document.getElementById("barWidthExample").style.width= width.toString() + '%';//update to our new width
																						   //and make sure it is treated as a percentage
	}
</script>
    Copy

    Displayed as:       

    Bar!

    More information
    document.getElementById("barWidthExample").style.width
    After the ".style" we can list any css attribute to view its current value or update it's current value. This is basically accessing the css attribute like a variable. The only caveat for accessing css attributes is that multi-word css attributes use a dash, but the javascript equivalent will instead use camelCase. Example - (CSS) font-size becomes (JS) fontSize
    width.replace('%', '')
    Many CSS attributes contain text & numeric information. To work with the numeric part, we'll need to seperate our numbers and text. An easy option is to replace the characters we don't want with empty space ''. Once we have the extra characters removed, we can modify the numeric part of our attribute. Some other text values you may need to remove include 'px', 'em', '#'.
    width.toString() + '%'
    Now that we have finished updating our width's value, we need to convert it back to a usable percentage. We use the .toString() method to treat our variable now as a piece of text and append the "%". We have now successfully converted our text to a number and then back to text!

    Accessing Tables

    It is possible to step through the individual cells of a table using some handy Javascript code! Here we'll learn how to step through the table's row & columns. 

    <button class="btn btn-primary" onclick="practiceTable()">Update Table</button>
<table class="table table-striped" id="p_table">
	<tr>
		<td>0</td><td>1</td><td>2</td>
	</tr>
	<tr>
		<td>3</td><td>4</td><td>5</td>
	</tr>
	<tr>
		<td>6</td><td>7</td><td>8</td>
	</tr>
</table>
<script>
	function practiceTable(){
		var table = document.getElementById("p_table");//Retrieve our table element
		var row_counter;//Keeps track of our row index
		var col_counter;//Keeps track of our column index
		var cell_value;
		for(row_counter = 0; row_counter < table.rows.length; row_counter++)
		{//Outer for loop iterates over each row
			for(col_counter = 0; col_counter < table.rows[row_counter].cells.length; col_counter++)
			{
				cell_value = table.rows[row_counter].cells[col_counter].innerHTML;//Read in a cells current value
				cell_value = parseInt(cell_value) + 2;//Increase the cell's value by 2
				table.rows[row_counter].cells[col_counter].innerHTML = cell_value;//Update the actual html of the cell
			}
		}
	}
</script>
